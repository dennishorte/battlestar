# Makers (Phase 4)

## Rules

- In this phase, spice accumulates on Maker board spaces.
- Check each board space with a Maker icon: Deep Desert, Hagga Basin, and Imperial Basin.
- If a Maker space does not have an Agent on it, place 1 spice from the bank on that space (in the spot designated for bonus spice).
- This spice is added to any bonus spice that may already be there from previous rounds.
