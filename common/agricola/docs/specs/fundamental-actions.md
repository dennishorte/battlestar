# Fundamental Actions

These are the atomic sub-actions that action spaces can offer. An action space may combine multiple fundamental actions (e.g. "Grain Utilization" offers both Sow and Bake Bread).

## Land Development

| Action | Description |
|---|---|
| **Plow field** | Convert one empty farmyard space into a field |
| **Sow** | Place grain/vegetable from supply onto empty fields |
| **Build fences** | Place wood fences to create pastures (1 wood each) |
| **Build stable(s)** | Place a stable on a farmyard space (2 wood) |
| **Build room(s)** | Add a room to the house (5 building material + 2 reed) |
| **Renovate house** | Upgrade house material (wood->clay->stone, costs 1 reed + 1 material/room) |

## Family

| Action | Description |
|---|---|
| **Family growth (requires room)** | Add a family member if an empty room is available |
| **Family growth (no room)** | Add a family member without needing an empty room |

## Cards

| Action | Description |
|---|---|
| **Play occupation** | Play an occupation card from hand (cost varies by player count) |
| **Play minor improvement** | Play a minor improvement card from hand |
| **Play major improvement** | Build a major improvement from the shared supply |

## Conversion

| Action | Description |
|---|---|
| **Bake bread** | Convert grain to food using an oven improvement |

## Other

| Action | Description |
|---|---|
| **Become starting player** | Take the starting player token |
