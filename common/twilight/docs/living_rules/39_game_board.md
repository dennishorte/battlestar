# 39 Game Board

The game board consists of all system tiles in play.

- **(39.1)** The game board consists of all system tiles that were placed during setup, even if the sides of those tiles do not touch any other system tiles, such as the Ghosts of Creuss' home system.

- **(39.2)** A system tile is on the edge of the game board if any of its sides are not touching another system tile.
    - a. The Ghosts of Creuss home system and the wormhole nexus are on the edge of the game board.
