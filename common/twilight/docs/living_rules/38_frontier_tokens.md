# 38 Frontier Tokens

Frontier tokens can be explored for a variety of game effects.

- **(38.1)** Frontier tokens are placed on the game board during setup. One frontier token is placed in each system that does not contain any planets.
    - a. Frontier tokens are not placed on hyperlane tiles.
    - b. A system cannot have more than one frontier token.
    - c. Frontier tokens are placed in anomalies that do not have planets.
    - d. A frontier token is placed in the Creuss gate system.
