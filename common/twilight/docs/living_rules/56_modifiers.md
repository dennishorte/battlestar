# 56 Modifiers

A modifier is a number that is applied by an ability to increase or decrease the attribute values of a unit or the results of a die roll.

- (56.1) A modifier is always preceded by the word "apply" followed by a numerical value.

- (56.2) A modifier value preceded by a "+" is added to the attribute or result being modified; a modifier value preceded by a "-" is subtracted from the attribute or result being modified.
