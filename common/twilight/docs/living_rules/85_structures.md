# 85 Structures

A structure is a type of unit. PDS units and space docks are structures.

- (85.1) Structures are always placed on planets.
    - a. The Clan of Saar's "Floating Factory" faction-specific space dock is placed in a system's space area.

- (85.2) Structures are primarily placed on planets using the "Construction" strategy card.

- (85.3) Structures cannot move or be transported.

- (85.4) A player can have a maximum of one space dock on each planet.

- (85.5) A player can have a maximum of two PDS units on each planet.

- (85.6) A player cannot place a structure on a planet if it would exceed the maximum number of allowed structures of that type on that planet.
