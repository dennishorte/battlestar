# Secret Objectives

Worth 1 VP each. 40 cards total (20 base + 13 PoK + 7 PoK status + 2 PoK agenda = 40... see breakdown below).

- Setup: Each player draws 2, keeps 1, returns the other to the deck (reshuffled).
- Max 3 secret objectives per player (scored + unscored). If you draw a 4th, return 1 unscored to the deck.
- Unlike public objectives, you do NOT need to control all home system planets to score these.

Omega (Ω) versions from the Codex replace the original text.

---

## Action Phase (12 cards)

Score immediately after fulfilling the requirement. Multiple action phase secrets can be scored per round, but each must be in a separate combat. A space combat and subsequent ground combat in the same tactical action count as separate combats.

### Base Game (5)

| ID | Name | Condition |
|----|------|-----------|
| destroy-their-greatest-ship | Destroy Their Greatest Ship | Destroy another player's war sun or flagship. |
| make-an-example-of-their-world | Make an Example of Their World | Ω: Destroy the last of a player's ground forces on a planet during the bombardment step. |
| spark-a-rebellion | Spark a Rebellion | Win a combat against a player who has the most victory points. |
| turn-their-fleets-to-dust | Turn Their Fleets to Dust | Ω: Destroy the last of a player's non-fighter ships in the active system during the space cannon offense step. |
| unveil-flagship | Unveil Flagship | Win a space combat in a system that contains your flagship. You cannot score this objective if your flagship is destroyed in the combat. |
